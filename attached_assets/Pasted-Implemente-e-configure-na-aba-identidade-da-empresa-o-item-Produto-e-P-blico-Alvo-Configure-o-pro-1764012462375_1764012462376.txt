3. Regra de validação obrigatória de valor mínimo
Para toda mídia, o aluno não pode inserir nenhum valor promocional inferior ao custo unitário mínimo.
Lógica obrigatória
Ao receber valor_inserido no formulário:
valor_inserido >= custo_unitario_minimo
Se for menor → bloquear.
Mensagem clara ao aluno
O valor informado é inferior ao mínimo permitido para esta mídia.
Valor mínimo: R$ X,XX — valor inserido: R$ Y,YY.
Consulte o Guia de Mídias no sistema.
4. Código para backend (exemplo em Python)
def validar_valor_midia(id_midia, valor_inserido, conn):
    query = """
        SELECT nome_midia, custo_unitario_minimo 
        FROM midias 
        WHERE id_midia = %s
    """
    midia, minimo = conn.fetchrow(query, id_midia)

    if valor_inserido < minimo:
        raise ValueError(
            f"O valor inserido para '{midia}' é inferior ao mínimo permitido (R$ {minimo:.2f})."
        )
5. Trigger de segurança no PostgreSQL
CREATE OR REPLACE FUNCTION trg_validar_midia()
RETURNS trigger AS $$
DECLARE
    minimo NUMERIC;
BEGIN
    SELECT custo_unitario_minimo 
    INTO minimo 
    FROM midias 
    WHERE id_midia = NEW.id_midia;

    IF NEW.valor_inserido < minimo THEN
        RAISE EXCEPTION 
        'Valor inválido: o valor para esta mídia (R$ %) não pode ser inferior ao mínimo (R$ %).',
        NEW.valor_inserido, minimo;
    END IF;

    RETURN NEW;
END;
$$ LANGUAGE plpgsql;

CREATE TRIGGER validar_midia_before_insert
BEFORE INSERT OR UPDATE ON decisoes_promocao
FOR EACH ROW EXECUTE FUNCTION trg_validar_midia();
6. Regra adicional obrigatória
Se o aluno tentar "fatiar" o valor de uma mídia (ex.: Outdoor custa R$ 2.000 e ele tenta lançar “500”), o sistema não deve aceitar, porque:
Cada linha do Guia representa um formato fechado
→ Não é permitido subdividir custo nem inserir valores parciais.
7. Vinculação pedagógica
O sistema deve reforçar ao aluno que:
O P de Promoção é baseado em custos reais. Cada mídia possui um preço mínimo definido no mercado e o simulador segue fielmente o Guia de Mídias.
